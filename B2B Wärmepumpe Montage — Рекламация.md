# **B2B Wärmepumpe Montage — Рекламация**

## **СТАДИЯ: РЕКЛАМАЦИЯ — R**

## **R1. Новая рекламация**

**Ответственный:** CRM / автоматическое действие системы

**Клиент:**

* заполняет форму рекламации по ссылке;
* указывает фирму / компанию;
* указывает контактное лицо;
* указывает телефон / email;
* указывает контакт владельца или ответственного на объекте;
* указывает адрес объекта;
* описывает причину проблемы;
* добавляет описание проблемы по пунктам;
* загружает фото / видео проблемы;
* нажимает кнопку «Отправить».

**CRM автоматически:**

* создает новую карточку в воронке «Рекламация»;
* переносит в карточку все данные из формы;
* добавляет фото / видео клиента;
* привязывает рекламацию к объекту;
* назначает ответственным Montageleiter;
* ставит статус «Новая рекламация».

**CRM чек-лист:**

* ⬜ Форма рекламации заполнена
* ⬜ Данные клиента внесены
* ⬜ Контакт ответственного на объекте внесен
* ⬜ Адрес объекта внесен
* ⬜ Описание проблемы добавлено
* ⬜ Фото / видео добавлены
* ⬜ Карточка рекламации создана
* ⬜ Ответственный Montageleiter назначен

**Критерий завершения:**

* карточка рекламации создана в CRM;
* все данные из формы перенесены;
* ответственным назначен Montageleiter.

**Далее:** R2. Владелец → Montageleiter.

## **R2. Технический анализ рекламации и решение**

**Ответственный:** Montageleiter

**Montageleiter:**

* анализирует рекламацию;
* если по фото / видео невозможно определить причину и обсчитать материалы и объём работ — организует дополнительную диагностику другой бригадой;
* заполняет Акт диагностики рекламации;
* ставит задачу Инженеру-ассистенту на расчёт материалов.

**АКТ ДИАГНОСТИКИ РЕКЛАМАЦИИ** (раскрывается кликом по фразе «заполняет Акт диагностики рекламации»)

* **Причина проблемы** — краткое описание причины рекламации.
* **Ответственность** — SkyBud; производитель; клиент; спорная.
* **Способ устранения** — краткое описание решения.
* **Бригада для выполнения работ** — бригада, которая должна выполнить исправление.
* **Трудозатраты** — количество людей; трудозатраты в часах; количество выездов.
* **Материалы** — спецификация материалов.
* **Стоимость материалов** — стоимость материалов.
* **Стоимость рекламации** — стоимость работ; стоимость материалов; итого.
* **Финансовый тип рекламации** — за счёт SkyBud; за счёт клиента; за счёт производителя; решение руководителя.
* **Заключение Montageleiter** — краткий итоговый вывод по рекламации.

**Инженер-ассистент:**

* рассчитывает стоимость материалов по спецификации Montageleiter;
* вносит данные в Акт диагностики рекламации.

**B2B-менеджер:**

* определяет финансовый тип рекламации;
* если рекламация не по вине SkyBud — готовит коммерческое предложение / счёт клиенту;
* если рекламация по вине SkyBud — относит стоимость рекламации в себестоимость объекта с пометкой «Расходы по рекламации».

**CRM чек-лист:**

* ⬜ Причина проблемы определена
* ⬜ Ответственность определена
* ⬜ Способ устранения определён
* ⬜ Трудозатраты рассчитаны
* ⬜ Акт диагностики заполнен
* ⬜ Спецификация материалов сформирована
* ⬜ Стоимость материалов рассчитана
* ⬜ Стоимость рекламации рассчитана
* ⬜ Финансовый тип рекламации определён
* ⬜ Коммерческое решение принято

**Критерий завершения:**

* техническое решение по рекламации сформировано;
* трудозатраты рассчитаны;
* спецификация материалов сформирована;
* стоимость материалов и стоимость рекламации рассчитаны;
* финансовое решение по рекламации принято.

**Далее:** R3. Владелец → B2B-менеджер.

## **R3. Согласование сроков устранения с клиентом**

**Ответственный:** B2B-менеджер

**B2B-менеджер:**

* переименовывает сделку, добавляя пометку «R»;
* ознакомляется с информацией, которую подготовил Montageleiter;
* если рекламация не по вине SkyBud — связывается с клиентом и готовит счёт;
* если рекламация по вине SkyBud — фиксирует стоимость затрат по рекламации;
* согласовывает дату выезда;
* согласовывает срок устранения рекламации;
* фиксирует договоренности в CRM;
* передает карточку Montageleiter в выполнение.

**CRM чек-лист:**

* ⬜ Дата выезда согласована
* ⬜ Срок устранения согласован
* ⬜ Договоренности с клиентом зафиксированы
* ⬜ Карточка передана Montageleiter в выполнение

**Критерий завершения:**

* дата и срок устранения согласованы с клиентом.

**Далее:** R4. Владелец → Montageleiter.

## **R4. Выполнение работ по устранению рекламации**

**Ответственный:** Montageleiter

**Montageleiter:**

* при необходимости пересогласовывает дату выполнения работ с клиентом;
* организует бригаду для исправления;
* передает бригаде задачу;
* добавляет ответственного исполнителя в группу объекта;
* передает способ решения;
* передает информацию о материалах;
* контролирует выполнение работ;
* фиксирует фактические трудозатраты;
* фиксирует фактический километраж;
* получает фото / видео после исправления;
* загружает фото / видео исправленной ошибки в карточку;
* передает карточку Ассистенту бухгалтера для расчёта себестоимости рекламации.

**Ассистент бухгалтера:**

* рассчитывает фактическую себестоимость рекламации;
* учитывает стоимость материалов, труда и амортизацию автомобиля / километраж;
* вносит расчёт себестоимости в карточку;
* передает карточку B2B-менеджеру для закрытия.

**CRM чек-лист:**

* ⬜ Бригада назначена
* ⬜ Задача на исправление передана
* ⬜ Ответственный исполнитель добавлен в группу объекта
* ⬜ Работы выполнены
* ⬜ Фактические трудозатраты внесены
* ⬜ Фактический километраж внесён
* ⬜ Фото / видео после исправления получены
* ⬜ Фото / видео загружены в карточку
* ⬜ Себестоимость рекламации рассчитана
* ⬜ Расчёт себестоимости внесён в карточку
* ⬜ Карточка передана B2B-менеджеру

**Критерий завершения:**

* ошибка исправлена;
* фото / видео исправления загружены в карточку;
* фактические затраты внесены;
* себестоимость рекламации рассчитана.

**Далее:** R5. Владелец → B2B-менеджер.

## **R5. Уведомление клиента и закрытие рекламации**

**Ответственный:** B2B-менеджер

**B2B-менеджер:**

* проверяет расчёт себестоимости рекламации;
* если рекламация по вине SkyBud — относит стоимость рекламации в себестоимость объекта;
* если рекламация не по вине SkyBud — связывается с клиентом;
* уведомляет клиента о выполнении работ;
* выставляет счёт клиенту;
* контролирует получение оплаты;
* фиксирует коммуникацию в CRM;
* закрывает карточку рекламации.

**CRM чек-лист:**

* ⬜ Стоимость рекламации отнесена в себестоимость объекта или выставлен счёт клиенту
* ⬜ Клиенту позвонили
* ⬜ Клиент уведомлен об исправлении
* ⬜ Фото / видео отправлены клиенту
* ⬜ Оплата получена (если рекламация платная)
* ⬜ Коммуникация зафиксирована в CRM
* ⬜ Карточка рекламации закрыта

**Критерий завершения:**

* клиент уведомлен;
* стоимость рекламации отнесена в себестоимость объекта или оплачена клиентом;
* Оплата получена (если рекламация платная);
* рекламация закрыта.
