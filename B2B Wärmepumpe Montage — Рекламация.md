# **B2B Wärmepumpe Montage — Рекламация**

## **СТАДИЯ: РЕКЛАМАЦИЯ — R**

## **R1. Новая рекламация**

**Ответственный:** CRM / автоматическое действие системы

**Клиент:**

* заполняет форму рекламации по ссылке;  
* указывает фирму / компанию;  
* указывает контактное лицо;  
* указывает телефон / email;  
* указывает контакт владельца или ответственного на объекте;  
* указывает адрес объекта;  
* описывает причину проблемы;  
* добавляет описание проблемы по пунктам;  
* загружает фото / видео проблемы;  
* нажимает кнопку “Отправить”.

**CRM автоматически:**

* создает новую карточку в воронке “Рекламация”;  
* переносит в карточку все данные из формы;  
* добавляет фото / видео клиента;  
* привязывает рекламацию к объекту;  
* назначает ответственным Montageleiter;  
* ставит статус “Новая рекламация”.

**CRM чек-лист:**

* ⬜ Форма рекламации заполнена  
* ⬜ Данные клиента внесены  
* ⬜ Контакт ответственного на объекте внесен  
* ⬜ Адрес объекта внесен  
* ⬜ Описание проблемы добавлено  
* ⬜ Фото / видео добавлены  
* ⬜ Карточка рекламации создана  
* ⬜ Ответственный Montageleiter назначен

**Критерий завершения:**

* карточка рекламации создана в CRM;  
* все данные из формы перенесены;  
* ответственным назначен Montageleiter.

**Далее:** R2. Владелец → Montageleiter.

**R2. Технический анализ рекламации и решение**

**Ответственный:** Montageleiter

**Montageleiter:**

• анализирует рекламацию;

• если по фото / видео невозможно определить причину, объём работ или перечень необходимых материалов — организует дополнительную диагностику другой бригадой;

• заполняет Акт диагностики рекламации.

**Далее:** R2.1. Владелец → Инженер-ассистент.

**R2.1. Расчёт стоимости материалов по рекламации**

**Ответственный:** Инженер-ассистент

**Инженер-ассистент:**

• проверяет спецификацию материалов;

• рассчитывает стоимость материалов по спецификации Montageleiter;

• вносит стоимость материалов в Акт диагностики рекламации;

• прикрепляет расчёт стоимости материалов к карточке рекламации;

• передает карточку B2B-менеджеру.

### **CRM чек-лист:**

⬜ Спецификация материалов проверена

⬜ Стоимость материалов рассчитана

⬜ Стоимость материалов внесена в Акт диагностики рекламации

⬜ Расчёт стоимости материалов прикреплён к карточке

⬜ Карточка передана B2B-менеджеру

### **Критерий завершения:**

• стоимость материалов рассчитана;

• стоимость материалов внесена в Акт диагностики рекламации;

• расчёт стоимости материалов прикреплён к карточке;

• карточка передана B2B-менеджеру.

**Далее:** R3. Владелец → B2B-менеджер.

**R3. Согласование сроков устранения с клиентом**

**Ответственный:** B2B-менеджер

**B2B-менеджер:**

• меняет название сделки с пометкой «R»;

• ознакомляется с информацией, которую подготовил Montageleiter;

• принимает финансовое решение по рекламации;

• если по рекламации предусмотрена оплата со стороны клиента — связывается с клиентом и готовит коммерческое предложение / счёт клиенту;

• согласовывает счёт с клиентом;

• если рекламация по вине SkyBud — фиксирует расходы по рекламации для дальнейшего отнесения в себестоимость объекта после расчёта фактической себестоимости;

• согласовывает дату выезда;

• согласовывает срок устранения рекламации;

• фиксирует договорённости в CRM;

• передает карточку Montageleiter в выполнение.

### **CRM чек-лист:**

⬜ Финансовое решение по рекламации принято

⬜ Коммерческое решение принято

⬜ Дата выезда согласована

⬜ Срок устранения согласован

⬜ Договорённости с клиентом зафиксированы

⬜ Карточка передана Montageleiter в выполнение

### **Критерий завершения:**

• финансовое решение по рекламации принято;

• дата выезда согласована;

• срок устранения рекламации согласован;

• договорённости зафиксированы в CRM;

• карточка передана Montageleiter в выполнение.

**Далее:** R4. Владелец → Montageleiter.

**R4. Выполнение работ по устранению рекламации**

**Ответственный:** Montageleiter

**Montageleiter:**

• при необходимости пересогласовывает дату выполнения работ с клиентом;

• организует бригаду для исправления;

• передает бригаде задачу;

• добавляет ответственного исполнителя в группу объекта;

• передает информацию о материалах;

• контролирует выполнение работ;

• фиксирует фактические трудозатраты в Акте диагностики рекламации;

• фиксирует фактический километраж в Акте диагностики рекламации;

• загружает фото / видео выполненных работ по устранению рекламации в карточку;

• передает карточку Ассистенту бухгалтера для расчёта фактической себестоимости рекламации.

### **CRM чек-лист:**

⬜ Бригада назначена

⬜ Задача на исправление передана

⬜ Ответственный исполнитель добавлен в группу объекта

⬜ Работы выполнены

⬜ Фактические трудозатраты внесены

⬜ Фактический километраж внесён

⬜ Фото / видео выполненных работ загружены в карточку

⬜ Карточка передана Ассистенту бухгалтера

### **Критерий завершения:**

• работы по устранению рекламации выполнены;

• фактические трудозатраты внесены;

• фактический километраж внесён;

• фото / видео выполненных работ загружены в карточку;

• карточка передана Ассистенту бухгалтера.

**Далее:** R4.1. Владелец → Ассистент бухгалтер.

# **R4.1. Расчёт фактической себестоимости рекламации**

**Ответственный:** Ассистент бухгалтер

**Ассистент бухгалтер:**

• рассчитывает фактическую себестоимость рекламации;

• учитывает стоимость материалов, труда и километража;

• вносит расчёт фактической себестоимости в карточку;

• передает карточку B2B-менеджеру.

### **CRM чек-лист:**

⬜ Фактическая себестоимость рекламации рассчитана

⬜ Расчёт себестоимости внесён в карточку

⬜ Карточка передана B2B-менеджеру

### **Критерий завершения:**

• фактическая себестоимость рекламации рассчитана;

• расчёт себестоимости внесён в карточку;

• карточка передана B2B-менеджеру.

**Далее:** R5. Владелец → B2B-менеджер.

**R5. Уведомление клиента и закрытие рекламации**

**Ответственный:** B2B-менеджер

**B2B-менеджер:**

• проверяет расчёт фактической себестоимости рекламации;

• если рекламация по вине SkyBud — относит стоимость рекламации в себестоимость объекта с пометкой «Расходы по рекламации»;

• если по рекламации предусмотрена оплата со стороны клиента:

 • связывается с клиентом;

 • уведомляет клиента о выполнении работ;

 • выставляет счёт клиенту;

 • контролирует получение оплаты;

• отправляет клиенту фото / видео выполненных работ по устранению рекламации;

• фиксирует коммуникацию в CRM;

• закрывает карточку рекламации.

### **CRM чек-лист:**

⬜ Финансовое решение исполнено

⬜ Клиент уведомлен о выполнении работ

⬜ Фото / видео отправлены клиенту

⬜ Оплата получена (если рекламация платная)

⬜ Коммуникация зафиксирована в CRM

⬜ Карточка рекламации закрыта

### **Критерий завершения:**

• клиент уведомлен о выполнении работ;

• финансовое решение по рекламации исполнено;

• оплата получена (если рекламация платная);

• карточка рекламации закрыта.
